Description of the contents of the submission

OpenStreetMapCaseStudy.pdf: answers to the rubric questions
AuditStreetNames.py: python script to audit street names

map.txt: description of the map region selected for data wrangling
san-jose.osm: a sample file of the region I chose. The original OSM file is 353 MB in size
references.txt: a list of references I used in this project
